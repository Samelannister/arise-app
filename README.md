# ARISE — Shadow Hunter System

Application RPG de productivité inspirée de Solo Leveling.

## Déploiement Vercel (2 minutes)

1. Crée un repo GitHub et push ce dossier
2. Va sur [vercel.com](https://vercel.com) → "Add New Project"
3. Importe ton repo GitHub
4. Vercel détecte Vite automatiquement → clique "Deploy"
5. Ton lien est prêt : `https://arise-xxx.vercel.app`

## Déploiement local

```bash
npm install
npm run dev
```

## Stack
- React 18 + Vite
- Web Audio API (sons génératifs)
- localStorage (données par utilisateur)
- PWA-ready (notifications)
